PHPUnit with PEAR Deprecated

Aug 15th, 2014

So I installed PHPUnit today with PEAR and apparently that wasn't what I should have done because the PHPUnit PEAR installing method has been officially deprecated.

If you did what I did and installed PHPUnit with PEAR you can easily uninstall it with:

pear uninstall phpunit/PHPUnit

And then install it with PHAR:

wget https://phar.phpunit.de/phpunit.phar
chmod +x phpunit.phar
mv phpunit.phar /usr/local/bin/phpunit

You might need to restart your terminal for the phpunit command to work.
